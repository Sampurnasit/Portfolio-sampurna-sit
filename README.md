# Sampurna Sit — Software Developer Portfolio

A premium, interactive, and high-performance personal portfolio website built with modern web technologies. This project showcases my journey as a **Software Developer & AI/ML Enthusiast**, highlighting my skills in Data Structures, Algorithms, and Core Engineering.

## 🚀 Live Demo
Experience the interactivity here: [sampurnasit.github.io](https://sampurnasit.github.io/) *(Note: Replace with your actual GitHub Pages URL if hosted elsewhere)*

## ✨ Key Features
- **Software Developer Focus**: Tailored bio, skills, and stats (100+ LeetCode solved) for SDE internship roles.
- **Interactive UI/UX**: Includes a custom cursor, smooth parchment-style reveal animations, and a blooming petal background effect.
- **Dynamic Projects Section**: Interactive "projects.md" tab featuring Attend-Ease AI, Personal Savings Manager, and Tic-Tac-Toe.
- **Resume Integration**: Direct link to download/view the latest professional resume.
- **Contact Interactivity**: Fully functional `mailto:` and `tel:` links for easy recruitment.
- **Responsive Design**: Optimized for both mobile and desktop experiences.

## 🛠️ Tech Stack
- **Languages**: HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Typography**: Google Fonts (Cormorant Garamond, DM Mono, Playfair Display).
- **Icons**: Emoji-based and custom SVG accents for a clean, semantic look.

## 🔧 How to Run Locally
1. **Clone the repository**:
   ```bash
   git clone https://github.com/sampurnasit/portfolio.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd portfolio
   ```
3. **Open the site**:
   Simply open `index.html` in any modern web browser.
